# Disciplinas

Las **Disciplinas** son poderes sobrenaturales otorgados por la Sangre de Caín. Cada clan se especializa en ciertas disciplinas, pero con el tiempo y el estudio, un vástago puede aprender otras.

## Lista de Disciplinas Comunes

- **Animalismo**: Dominio sobre la bestia interior y los animales.
- **Auspex**: Percepción sobrenatural.
- **Celeridad**: Velocidad sobrenatural.
- **Dominación**: Control mental sobre otros.
- **Fortaleza**: Resistencia sobrenatural.
- **Obtenebración**: Control de las sombras (Clan Lasombra).
- **Potencia**: Fuerza sobrenatural.
- **Presencia**: Atracción y temor sobrenatural.
- **Protean**: Metamorfosis y adaptación (Clan Gangrel).
